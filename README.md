# Weather Diary (Дневник погоды)

## Автор
Молчанов Кирилл
## Описание
Weather Diary — это простое GUI-приложение на Python, предназначенное для ведения дневника наблюдений за погодой. 
Приложение позволяет пользователю фиксировать ежедневные показатели погоды, просматривать историю записей в удобном табличном формате и фильтровать данные по дате или температуре.

Все данные сохраняются локально в формате JSON, что обеспечивает простоту переноса и резервного копирования.

## Функциональные возможности
- Добавление записей: Ввод даты, температуры, текстового описания и отметки об осадках.
- Валидация данных: Проверка формата даты (ДД.ММ.ГГГГ), числового значения температуры и наличия описания.
- Просмотр: Отображение всех записей в таблице с возможностью прокрутки.
- Фильтрация: 
  - Поиск записей за конкретную дату.
  - Выбор записей с температурой выше указанного порога.
- Сохранение: Автоматическое сохранение всех изменений в файл `weather_data.json`.

## Технологии
- Python 3.x
- Tkinter (стандартная библиотека GUI)
- JSON (для хранения данных)
- Git (система контроля версий)

## Как запустить приложение

1. Убедитесь, что у вас установлен Python версии 3.6 или выше.
2. Клонируйте репозиторий или скачайте исходный код:
   git clone https://github.com/YOUR_USERNAME/weather-diary.git
   cd weather-diary
